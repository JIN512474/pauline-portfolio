# pauline-portfolio-sample

Next.js(App Router) + Tailwind 기반 샘플 포트폴리오.

## 1) 로컬 실행
```bash
npm i
npm run dev
```
- http://localhost:3000

## 2) 메인 영상 교체
`/public/hero.mp4` 파일을 넣으면 홈 메인에 자동 재생됩니다.
- 권장: H.264 mp4, 10~20초, 1080p

## 3) 포트폴리오 이미지 교체
`/public/ph/01.jpg` ~ `06.jpg` 교체.

## 4) 배포 (Vercel)
- GitHub에 올린 뒤 Vercel에서 Import → Deploy
